# 80s Style RPG Game

This project is a simple 80s-style RPG game created using Pygame. The game features a tile-based world, character movement, and a basic structure for expanding into a more complex RPG.

## Project Structure

## Features

- **Tile-Based Graphics**: The game world is represented using a grid of tiles.
- **Player Movement**: The player can move around the map using the arrow keys.
- **Simple Graphics**: The game uses simple 2D sprites for the player and tiles.

## Getting Started

### Prerequisites

- Python 3.312
- Pygame library

### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/annd0/Aarcus.git
    cd 80s-style-rpg-game
    ```

2. **Install Pygame:**

    ```bash
    pip install pygame
    ```

### Running the Game

To run the game, simply execute the `main.py` script:

```bash
python main.py
